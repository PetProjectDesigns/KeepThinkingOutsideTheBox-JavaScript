Function Parameters and Return Values

To make functions truly useful, they need a way to receive data (input) and a way to send processed data back (output). This is achieved through parameters and the return statement.

Analogy: A function is like a calculator button. You give it numbers (parameters/input), it calculates (function body), and it displays the final answer (return value/output).

The Example Code: A Simple Adder

This function, named add, is defined to take two pieces of information (a and b) and use the return keyword to output their sum.

function add(a, b) // Parameters: a and b
{
    return a + b; // Return Value: The result of the calculation
}

// Calling the function and storing the result:
let result = add(5, 3); // Arguments 5 and 3 are passed in

// result is 8
console.log(result); // Outputs: 8

1. Parameters (Input)

  • Definition: Variables listed in the function's definition, like a and b in `function add(a, b)`.
  • Purpose: They act as placeholders for the values (arguments) that will be passed into the function when it is called.

2. The `return` Statement (Output)

  • Definition: The return keyword is used to stop function execution and specify a value to be sent back to the place where the function was called.
  • Effect: Once return is executed, the function immediately stops. The returned value can be stored in a variable (like result = add(5, 3)), or used directly in another expression.
