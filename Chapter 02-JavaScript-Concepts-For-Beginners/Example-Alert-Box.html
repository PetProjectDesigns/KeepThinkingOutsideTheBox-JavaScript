Creating a JavaScript Alert Box

The Alert Box is the simplest way for a JavaScript program to communicate with the user. It is a popup window that displays a message and waits for the user to click the "OK" button before continuing.

While modern websites rarely use alert() (because it stops the user from interacting with the page), it is an essential first step in understanding browser communication.

The `alert()` Syntax

The function is very straightforward. You simply pass the message you want to display inside the parentheses.

alert('Your message goes here!');
Note on Syntax: The word alert is the function name. The text inside the single quotes ('...') is a String (the message). The semicolon (;) ends the statement.

Key Takeaways

  • The alert() function is used to display a message in a modal popup.
  • It pauses the execution of the rest of the JavaScript code until the user clicks OK.
  • The message must be passed as a String (text inside quotes).
