Creating a JavaScript Prompt Box

The Prompt Box is a key way to make your JavaScript program interactive. Unlike the simple alert(), the prompt box allows the user to enter text, which your program can then store and use. This is the first step toward handling user input!

The prompt() and alert() combination

The most important concept here is Variable Assignment. Whatever the user types into the prompt box is immediately saved into a variable (in this case, userName).

let userName = prompt('What is your name?');
alert('Hello, ' + userName + '!');
How it Works: The program first asks the question using prompt(). Once you hit OK, your answer is saved in the `userName` variable. The second line uses String Concatenation (+) to combine the fixed text ("Hello, ") with your input and display it in an alert.

The Code Behind the Button

This is the exact code that runs when you click the button:

function runPromptExample() {
    let userName = prompt('What is your name?');
    
    // Check if the user entered anything (or clicked Cancel)
    if (userName) {
        alert('Hello, ' + userName + '! Welcome to the world of interactive JavaScript.');
    } else {
        alert('No problem! You can try again anytime.');
    }
}

Key Takeaways

  • The prompt() function gets user input and returns it as a String.
  • The return value must be stored in a variable (let userName = ...).
  • If the user clicks "Cancel" or closes the prompt, the variable will be set to null (or an empty string in some browsers).
