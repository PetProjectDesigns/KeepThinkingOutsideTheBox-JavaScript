Understanding JavaScript Operators

Operators are the symbols that tell the JavaScript engine to perform a specific action, whether it's calculating a sum, comparing two values, or saving data to a variable. They are the verbs of your program.

1. Arithmetic Operators

Arithmetic operators perform basic mathematical calculations. This is how you handle numbers in JavaScript, just like on a calculator.

An Analogy: Your Calculator: Think of arithmetic operators as basic tools in a calculator. They help you perform essential mathematical operations.
Operator Name & Function Code Example Result
+ Addition: Adds two numbers. let sum = 5 + 3; // sum is 8
- Subtraction: Subtracts one number from another. let difference = 10 - 4; // difference is 6
* Multiplication: Multiplies two numbers. let product = 6 * 7; // product is 42
/ Division: Divides one number by another. let quotient = 20 / 4; // quotient is 5
% Modulus: Returns the remainder of a division. (Useful for checking odd/even numbers!) let remainder = 15 % 4; // remainder is 3

2. Relational and Logical Operators

These operators are fundamental to decision-making in your code (like checking if a user's score is high enough to win).

Relational Operators (Comparison)

Relational operators compare values and always return a Boolean result (true or false).

Operator Name & Function Code Example Result
== Equal to: Checks if values are equal. let isEqual = (5 == 5); // isEqual is true
!= Not equal to: Checks if values are not equal. let isNotEqual = (5 != 3); // isNotEqual is true
> Greater than: Checks if the left value is larger. let isGreater = (8 > 6); // isGreater is true
< Less than: Checks if the left value is smaller. let isLess = (3 < 7); // isLess is true

Logical Operators

Logical operators combine the results of two or more relational checks.

Operator Name & Function Code Example Result
&& AND: Both conditions must be `true` for the result to be `true`. let result = (5 > 3 && 8 < 10); // result is true
|| OR: At least one condition must be `true` for the result to be `true`. let result = (5 > 3 || 8 > 10); // result is true
! NOT: Inverts a condition. `true` becomes `false`, and `false` becomes `true`. let result = !(5 > 3); // result is false

3. Assignment Operators

Assignment operators are used to assign (save) a value to a variable. The simple `=` operator saves a value, while compound assignment operators (`+=`, `-=`, etc.) perform an operation and then save the result.

Operator Name & Function Code Example Value After Operation
= Equal: Assigns a value to a variable. let y = 5; // y is now 5
+= Add and assign: Adds a value to a variable and assigns the result. (Shorthand for `y = y + 3;`) let y = 5; y += 3; // y is now 8
-= Subtract and assign: Subtracts a value from a variable and assigns the result. (Shorthand for `z = z - 2;`) let z = 10; z -= 2; // z is now 8
