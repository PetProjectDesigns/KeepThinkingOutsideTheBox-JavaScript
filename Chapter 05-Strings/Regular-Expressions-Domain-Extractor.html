Domain Extraction

Introduction

Regular Expressions (RegEx) can be used to isolate specific parts of a string, such as the domain name of an email address.

Key Concept: We use a capturing group (...) in our RegEx pattern to target the text immediately following the @ symbol.

Code Demonstration

The function iterates through a list, applies a RegEx match, and cleans up any trailing punctuation.

function extractDomains(emailList) {
    const domains = [];
    // Matches domain after '@'
    const pattern = /@([\w.-]+\.\w+)/;

    for (const email of emailList) {
        const match = email.match(pattern);
        if (match) {
            // Remove trailing punctuation
            const domain = match[1].replace(/\.$/, "");
            domains.push(domain);
        } else {
            domains.push("Invalid email");
        }
    }
    return domains;
}

Key Takeaways

  • Capturing Groups: Use match[1] to access the specific part of the RegEx defined inside parentheses.
  • Data Cleaning: Use replace(/\.$/, "") to ensure domains don't end with unwanted punctuation.
  • Validation: Always provide a fallback (like "Invalid email") for cases where the pattern does not match.
