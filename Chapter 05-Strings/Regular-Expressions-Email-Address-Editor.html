Email Data Stripping

Introduction

Sometimes you need to clean up user data by removing specific characters or symbols. In this lesson, we use the .replaceAll() method to strip symbols from an email address.

Key Concept: String methods allow you to transform text without changing the original variable.

Code Demonstration

This function takes an email string and uses chaining to remove multiple parts at once.

function stripEmail(email) {
    // .replaceAll finds every instance and removes it
    return email.replaceAll('@', '').replaceAll('.com', '');
}

Method Comparison Table

Understanding the difference between replace and replaceAll is vital for data cleaning.

Method Definition Outcome
.replace() Replaces only the first match found. "test@test@.com" → "testtest@.com"
.replaceAll() Replaces every instance found in the string. "test@test@.com" → "testtest.com"

Key Takeaways

  • Method Chaining: You can stack multiple .replaceAll() calls on one line.
  • Immutability: The original email input remains unchanged; the method returns a new, cleaned string.
