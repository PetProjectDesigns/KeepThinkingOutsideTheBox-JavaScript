Inserting Values into Strings

Introduction

In JavaScript, the modern way to insert values into a string is called Template Interpolation.

Key Concept: You use backticks (``) instead of single or double quotes, and wrap your variables in ${ }.

Code Demonstration

Below is how we use variables inside a single string without needing multiple plus (+) signs.

const name = 'Jordan';
const age = 22;
const hobby = 'painting';

// The ${} syntax injects the variable value directly into the string
const message = `Hello ${name}, you are ${age} and like ${hobby}!`;

Comparison Table

Notice how much cleaner the Template Literal syntax is compared to old-school concatenation.

Concept Definition Example
Concatenation Joining strings with the + operator. "Hi " + name + "!"
Interpolation Inserting values directly with ${}. `Hi ${name}!`

Key Takeaways

  • Use backticks (``) to define the string.
  • Wrap your variables or expressions in ${}.
  • This method is much easier to read and prevents errors with spaces.
