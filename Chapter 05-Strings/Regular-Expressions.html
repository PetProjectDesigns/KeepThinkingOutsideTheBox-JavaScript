JavaScript RegEx

Introduction

Regular Expressions (RegEx) are patterns used to match character combinations in strings. In JavaScript, RegEx are also objects.

Key Concept: RegEx literals are enclosed in forward slashes (e.g., /pattern/). They allow for powerful searching, matching, and replacing of text.

Code Demonstration

There are three common ways to use RegEx in JavaScript: Testing for existence, Matching multiple items, and Replacing characters.

// 1. Testing a pattern
const pattern = /JavaScript/;
const text = "Hello, JavaScript!";
if (pattern.test(text)) { console.log("Pattern found!"); }

// 2. Global matching (Email Example)
const emailPattern = /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}\b/g;
const emails = textWithEmails.match(emailPattern);

// 3. Replacing characters (Censoring digits)
const replacedText = textWithNumbers.replace(/\d/g, "*");

Common RegEx Methods

JavaScript provides several built-in methods to work with Regular Expressions.

Method Definition Returns
test() Checks if a pattern exists in a string. Boolean (true/false)
match() Retrieves matches when matching a string against a pattern. Array of matches or null
replace() Replaces matched text with a new substring. A new string

Key Takeaways

  • Literals: Use / / for a RegEx literal.
  • Flags: The g flag at the end stands for "global," meaning it finds all matches instead of stopping at the first one.
  • Special Characters: \d is a shorthand that matches any digit (0-9).
