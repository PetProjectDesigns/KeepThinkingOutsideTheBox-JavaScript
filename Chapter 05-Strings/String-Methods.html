JS String Methods

Introduction

Strings in JavaScript are more than just text; they are objects with built-in methods that allow you to transform, slice, and analyze them easily.

Note: Strings are immutable. This means these methods don't change the original string; they return a brand new one.

Method Comparison Table

Method Purpose Example Output
.toUpperCase() Converts all letters to uppercase. "HELLO"
.toLowerCase() Converts all letters to lowercase. "hello"
.slice(start, end) Extracts a part of a string. "ell" (from index 1 to 4)
.replace(x, y) Replaces text with new text. Swaps words or characters.
.split(" ") Divides string into an array. ["Hello", "World"]

Code Demonstration: Capitalization

To capitalize the first letter, we combine .charAt(), .toUpperCase(), and .slice().

let str = "javascript";
// 1. Get first char and upper it
// 2. Add the rest of the string starting from index 1
let capitalized = str.charAt(0).toUpperCase() + str.slice(1);
console.log(capitalized); // "Javascript"
