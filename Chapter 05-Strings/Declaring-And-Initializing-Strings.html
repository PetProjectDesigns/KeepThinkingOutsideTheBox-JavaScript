Declaring and Initializing Strings

String Fundamentals: Quotes

A String is a sequence of characters used to represent text. In JavaScript, you can use three different types of delimiters to define a string:

Note: Single quotes (') and double quotes (") behave identically. The primary reason to use one over the other is to avoid conflicts when the string itself contains the other type of quote (e.g., '"It's easy"').

Standard String Declaration (Single and Double Quotes)

We use the let keyword to declare a variable and assign a string literal to it using either single or double quotes.

// Declaring and initializing strings
let myString = "Hello, world!";
let anotherString = 'JavaScript is fun! Keep thinking outside the box!';

Template Literals (Backticks)

Template literals, enclosed by backticks (`` ` ``), offer powerful features that standard strings lack, particularly support for multiline strings and variable substitution (interpolation).

// Multi-line string with template literal
let multiLineString = `This is a string
that
spans multiple
lines
in JavaScript.`;

console.log(multiLineString);

Essential String Methods

Strings come with built-in methods (functions) to easily manipulate and format text. Three of the most common are toUpperCase(), toLowerCase(), and trim(). They do not change the original string but return a new, modified string.

let text = "   hello, world!   ";

console.log(text.toUpperCase()); // HELLO, WORLD!
console.log(text.toLowerCase()); // hello, world!
console.log(text.trim());        // hello, world! (Whitespace removed)

String Quote Comparison

A summary of the differences between the three main ways to define a string in modern JavaScript.

Syntax Multliline Support Variable Interpolation
Double Quotes (`"`) No (requires `\n`) No
Single Quotes (`'`) No (requires `\n`) No
Backticks (`` ` ``) Yes (natively) Yes (using ${variable})

Key Takeaways

  • Strings are immutable: Methods like toUpperCase() return a new string; they do not modify the original variable.
  • Template literals (backticks) are the modern standard for text that spans multiple lines or requires dynamic values.
  • Always use trim() when handling user input to remove accidental leading or trailing spaces.
