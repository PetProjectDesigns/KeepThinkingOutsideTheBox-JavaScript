Loop Control: The `continue` Statement

The continue statement is used inside loops to immediately stop the current iteration and jump to the next one. This means any code after the continue statement within the loop body is skipped for that specific iteration.

Use Case: Imagine you are processing a list of items and some are invalid (e.g., empty strings, null values). You can use an if statement to check for invalidity and then continue to avoid running complex, time-consuming code on that invalid item, saving processing time.

Using continue to Filter Numbers

In this example, the loop counts from 0 to 9. We use an if condition to identify even numbers (i % 2 === 0) and then use continue to skip the rest of the loop body (the console.log statement), effectively printing only odd numbers.

for (let i = 0; i < 10; i++) {

    if (i % 2 === 0) {
        // Skip even numbers (0, 2, 4, 6, 8)
        continue;
    }
    
    console.log('Odd number: ' + i); // This line runs only for 1, 3, 5, 7, 9
}

// The output will show only the odd numbers.

Key Takeaways

  • The continue statement skips only the current iteration , unlike break, which exits the entire loop.
  • It moves the flow immediately to the loop's update expression (e.g., i++ in a for loop) and then re-checks the condition.
  • Use continue for filtering or prematurely avoiding work on elements that don't meet a specific requirement.
