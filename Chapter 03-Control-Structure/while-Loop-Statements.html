The `while` Loop Statement

The while loop is a powerful control flow statement that repeats a block of code, as long as its condition evaluates to true. Unlike the `for` loop, which is often used for a fixed number of iterations, the while loop is best when you don't know exactly how many times the loop needs to run.

Analogy: Think of doing laundry. You use a while loop: WHILE there are dirty clothes in the basket, DO a load of laundry. When the condition (dirty clothes > 0) becomes false, the loop stops.

The Structure of the `while` Loop

A while loop requires two key elements outside of the while keyword itself:

let count = 0; // 1. Initialization (Outside the loop)

while (count < 5) // 2. Condition (Checked before each run)
{
    console.log('Count: ' + count);
    
    count++; // 3. The essential Increment/Updater (Inside the loop body)
}

Key Parts Explained:

  • 1. Initialization ({let \ count = 0}): You must define the counter/state variable before the loop starts.
  • 2. Condition ({count < 5}): The loop continues running only while this expression is true.
  • 3. Updater ({count++}): Crucially, you must include a statement inside the loop body that eventually makes the condition false. If you forget count++, the condition will always be true, creating an infinite loop!

The Code We Are Running

This function executes the requested loop and displays the output:

function runWhileLoop() {
    let count = 0;
    let output = '';

    while (count < 5) {
        output += ('Count: ' + count + '\n');
        count++;
    }
    document.getElementById('result-output').innerText = output;
}

Key Takeaways

  • Use while loops when you don't know the number of iterations beforehand (e.g., looping until a user enters a valid password).
  • Initialization happens before the loop.
  • The Updater/Increment must be inside the loop body to prevent infinite loops.
