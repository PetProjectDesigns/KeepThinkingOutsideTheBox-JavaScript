If and Else Conditional Statements

Conditional statements allow you to execute code based on certain conditions. The if statement checks a condition, and if it's true, the code block inside the if statement is executed. If it's false, the code block inside the else statement (if provided) is executed.

Analogy: Think of a traffic light. IF the light is green, THEN you drive. ELSE (if it's red or yellow), you wait. This is the core logic of if/else.

The `if/else` Syntax

The condition is placed in parentheses after the if keyword. The code to run is placed inside curly braces {}.

let age = 20;

if (age >= 18)
{
    console.log('You are an adult.');
}
else {
    console.log('You are a minor.');
}

// Result of this code: 'You are an adult.'

The Code Behind the Checker

This function takes the value you type in, compares it against 18, and uses if/else to determine the output message:

function checkAge() {
    let age = document.getElementById('user-age').value;

    if (age >= 18) {
        // Condition is TRUE
        document.getElementById('result-output').innerText = "TRUE: You are 18 or older and an adult.";
    } else {
        // Condition is FALSE
        document.getElementById('result-output').innerText = "FALSE: You are a minor.";
    }
}

Key Takeaways

  • The if statement requires a condition inside the parentheses (e.g., age >= 18).
  • The condition must evaluate to a Boolean (true or false).
  • The else block is optional and only runs if the if condition is false.
